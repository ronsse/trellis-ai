# `eval/datasets/`

Hand-curated fixtures small enough to commit (a few KB to a few MB
total). Larger corpora are generated by `eval/generators/` or fetched
at runtime — never committed.

When you add a dataset, document its provenance here:

- **Source** — where it came from (synthesised? captured? scraped?
  from which run?).
- **Schema** — what fields are present and what do they mean.
- **Why this size** — what scenario uses it and why a smaller fixture
  isn't enough.
